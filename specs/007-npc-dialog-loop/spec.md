# Spec 007: NPC Interaction and Dialog UI

## Goal

Extend the playable loop with in-world NPC interaction and an on-canvas dialog box.

## Acceptance Criteria

- AC-01: Town scene renders at least one NPC marker.
- AC-02: When player is near an NPC, HUD/canvas shows an interaction hint.
- AC-03: Pressing `e` near NPC opens dialog UI with choices.
- AC-04: Pressing `1` or `2` selects a dialog choice and closes dialog.
- AC-05: Selecting a faction choice updates displayed faction state.
- AC-06: Existing title/town/battle loop remains functional.
